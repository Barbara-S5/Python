1. print - это встроенная в Python подпрограмма для вывода данных на экран
2. = - Переменная, сохраняющая значение месседжа, после равно значение идет в кавычках
 Message = "Guten Tag"
 print(Message)
 Но Числовые значения без кавычек
 TF = 25
3. str() - Преобразованием данных из одного типа в другой (для превращения числа в строку)
4. int() - преобразовать строку в целое число
   - Дробные числа приводят к целым (не округляет числа по правилам арифметики, а просто отбрасывает дробную часть)
5. Десятичные числа пишутся через обычную точку
6. float()- преобразовать строку в дробное число
